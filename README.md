A sight words learning app for 6-year-olds, built with research-backed learning science.

## Features

- **50 Fry high-frequency sight words** with Heart Word phonics integration
- **Spaced repetition** using a 4-box Leitner system
- **Touch-optimized** for phones and tablets
- **Engaging theme**: Plant eggs ‚Üí Nurture with practice ‚Üí Hatch creatures!
- **Text-to-speech** for all words
- **Progress saved locally** in browser

## How It Works

1. **New words** plant mystery eggs in your garden
2. **Reviews** help eggs grow and crack
3. **Mastery** (3 correct in a row) hatches a creature!
4. Collect creatures by learning more words

## Learning Science

Based on:
- SM-2 / Leitner spaced repetition
- Heart Word methodology (phonics integration)
- 85% success rate target
- 5-8 minute sessions
- Testing effect / active recall
